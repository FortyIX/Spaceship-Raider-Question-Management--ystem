question uploader v1.0  

------------------------------------------------

 

                          

       Instructions                                                

      1.click the "select" button and choose a question image

      2.type the correct answer for this question

      3.select the difficulty for that question

      4 click upload button to start uploading 

      5 logoff the account (very important)

                                                  

 

      Specification required of question image                              

 

  • Size of image must be less than 500 Kilobytes

  • Questions in the image has to be clear enough to see on mobile phone  

  • only one question allowed to appear on each image

  • questions must be able to produce an integer/expression result  

  • questions must be able to solved in no longer than 5 mins 

   

     uploading requirement

  •   name of the image file has to be understandable                                                    

     Answer requirement 

         Answers must be one of the following formats:

  •     Integer                             e.g   123,865,10,cos(pi),cos(2pi)

  •     simple expression                   e.g    3x,7x,6x+1,4x+8,9i+5,6i+8j 

  •     complex expression                 e.g    3x^2,3/2*x,3*x^-1

 

      Range of math question

  •   Basic Calculation

  •   Trigonometry 

  •   vector 

  •   calculus

  •   diffierential equations

  •   complex number

  •   inequlity          

 

     Suggestions 

  •   collect existing questions and modify them 

  •   answer come with root is not recommended
